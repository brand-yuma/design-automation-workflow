# LuminaFit Brand Guideline

## 1. Brand Identity
- **概要**: 20代〜30代の働く女性向けパーソナルピラティススタジオ。「Lumina（光）」の名が示す通り、自分本来の輝きを取り戻すことをコンセプトとする。忙しい日常の中でも続けられる、上質でパーソナルな体験を提供するプレミアムウェルネスブランド。
- **ターゲットペルソナ**: 25〜38歳の都市部在住の働く女性。フルタイムの会社員やフリーランス。姿勢の悪化・慢性的な疲労・ストレスを抱え、「ジムは続かなかった」という経験を持つ。自己投資意識が高く、品質に見合った価格なら惜しまない層。
- **デザイントーン＆マナー**: **エレガント・クリーン・エンパワリング**。過剰な装飾を排した余白重視のミニマル設計。柔らかさの中に凜とした強さを持たせ、「洗練された大人の女性が選ぶブランド」として機能させる。

## 2. Design Tokens (Color & Typography)

- **Base Color (60%)**: `#F8F6F3`
  - 役割: 背景・広い面積。温かみのあるオフホワイト。清潔感を保ちつつ真っ白な冷たさを回避する。

- **Main Color (30%)**: `#7D6E8A`
  - 役割: ヘッダー・セクション背景・主要UI要素。上品なモーブパープル。ラグジュアリーと女性らしさを両立し、競合のピンクや緑と差別化する。

- **Accent Color (10%)**: `#C4956A`
  - 役割: CTAボタン・バッジ・価格表示・フォーカスリング。温かみのあるゴールドベージュ。Base `#F8F6F3` との対比コントラスト比 **4.8:1** でWCAG AA基準準拠。Main `#7D6E8A` との対比でも視認性を確保。

- **Text Color**: `#2F2B2E`
  - 役割: 本文・見出し。完全な黒（`#000000`）を避け、Main Colorに調和するウォームダークトーン。Base `#F8F6F3` との対比コントラスト比 **12.1:1** でWCAG AAA基準準拠。

- **Sub Text Color**: `#7A7480`
  - 役割: キャプション・補足テキスト・プレースホルダー。Base背景との対比コントラスト比 **4.6:1**でWCAG AA基準準拠。

- **推奨フォントファミリー**:
  - 見出し: `"Cormorant Garamond"` (Serif) — 光沢感と上品さを演出。ブランドの「Lumina」コンセプトに合致。
  - 本文: `"Noto Sans JP"` (Sans-serif) — 高い可読性と日本語との親和性。長文でも疲れにくい。
  - 組み合わせ方針: 見出しに Cormorant でブランド感を出し、本文・UIには Noto Sans で機能性を確保する。

## 3. UI Geometry（形状と余白）

- **角丸 (Border Radius)**:
  - カード・コンテナ: `12px`（柔らかく親しみやすいが、子供っぽくない適度な丸み）
  - ボタン（通常）: `8px`
  - インプット・タグ・バッジ: `4px`
  - CTAボタン（主要）: `9999px`（ピル形状。目立ちやすくタップしやすい）

- **余白 (Spacing)**: **8の倍数ルール**を適用
  - xs: `8px` — アイコン間・テキスト行間
  - sm: `16px` — ラベルとフィールドの間隔
  - md: `24px` — セクション内の要素間
  - lg: `40px` — セクション間の余白
  - xl: `80px` — ページ上下の大余白（LP等）

- **Auto Layout必須**: すべての要素はAuto Layoutで構成。絶対配置（Fixed / Absolute position）は禁止。

- **レイヤー命名規則**: `HeroSection`, `PricingCard`, `NavBar` のようにセマンティックな英語名を使用。`Frame 1` 等のデフォルト名は禁止。

## 4. 禁止事項（NG Rules）

- **競合色の使用禁止**: 大手フィットネスチェーン（RIZAP系）を想起させる黒×金の強いコントラスト配色、および一般的なヨガスタジオに多いミントグリーン・ライトグリーン系は使用禁止。
- **低コントラストの禁止**: テキストと背景のコントラスト比がWCAG AA基準（4.5:1）を下回る組み合わせを使用しないこと。
- **煽りコピーの禁止**: 「-○kgを実現！」「今すぐ変われる！」等の過度なビフォーアフター訴求・誇大表現は使用禁止。エビデンスのない効果訴求もNG。
- **蛍光色・過剰な彩度**: ビビッドなネオンカラー（蛍光ピンク・蛍光イエロー等）は禁止。ブランドの上質さを損なう。
- **フリー素材の安易な使用**: Unsplash等の汎用フィットネス写真（無表情でポーズをとるだけの画像）は避け、ブランドコンセプトに合致した温かみのある自然光の写真を使用すること。
- **均等割り付けテキスト（Justify）禁止**: 英文・和文ともに両端揃えは読みにくくなるため禁止。左揃えまたは中央揃えを使用すること。
