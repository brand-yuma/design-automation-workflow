# Layout Rules — 336×280 (Large Rectangle)

## レイアウトの型
**拡張Z型（画像の上にコンテンツカードを重ねる）**

背景全面を画像で埋め、その上に半透明カードを absolute で下部に固定する。
このサイズのみ `absolute` を例外的に使用する（仕様上の設計）。

---

## ルートコンテナ

```jsx
{/* ⚠️ このサイズのみ relative を使用。absolute は背景画像とカードのみに限定する */}
<div className="w-[336px] h-[280px] flex flex-col overflow-hidden relative">
```

---

## 背景（フル画像）

```jsx
{/* BackgroundImage — absolute inset-0 で全面配置 */}
<img
  src="data:image/jpeg;base64,..."
  alt=""
  className="absolute inset-0 w-full h-full object-cover"
/>
```

- `absolute inset-0` でルートコンテナ全面を覆う
- `object-cover` でアスペクト比を維持しながらトリミング
- Base64データURIを使用。外部URLは禁止

---

## コンテンツカード（下部60%に固定）

```jsx
{/* ContentCard — absolute bottom-0 で下部固定 */}
<div className="absolute bottom-0 w-full h-[60%] bg-white/95 flex flex-col justify-between p-5 rounded-t-xl">

  {/* HeadlineBlock */}
  <div className="flex flex-col gap-1.5">
    <p className="text-[10px] font-bold tracking-widest uppercase"
       style={{color: 'var(--color-main)'}}>
      {eyebrow}
    </p>
    <h1 className="text-[20px] font-bold leading-tight"
        style={{fontFamily: 'var(--font-heading)', color: 'var(--color-text)'}}>
      {headline}
    </h1>
    <p className="text-[11px] leading-relaxed"
       style={{color: 'var(--color-subtext)'}}>
      {subCopy}
    </p>
  </div>

  {/* CTAButton — w-full で横幅いっぱい */}
  <button
    className="w-full flex flex-row items-center justify-center gap-2 py-3 text-[13px] font-bold text-white"
    style={{background: 'var(--color-accent)', borderRadius: 'var(--radius-btn)'}}
  >
    {ctaText}
    <svg width="13" height="13" viewBox="0 0 24 24" fill="none"
         stroke="currentColor" strokeWidth="2.5" strokeLinecap="round" strokeLinejoin="round">
      <line x1="5" y1="12" x2="19" y2="12"/>
      <polyline points="12 5 19 12 12 19"/>
    </svg>
  </button>

</div>
```

---

## 配置サマリー

| エリア | 配置 | 高さ | 役割 | AIBAC |
|---|---|---|---|---|
| BackgroundImage | absolute inset-0 | 100% | 感情的インパクト | Attention |
| ContentCard | absolute bottom-0 | 60% = 168px | 訴求 + CTA | Interest / Benefit / Action |

---

## `absolute` 使用の制限（このサイズ限定の例外）

```
許可: BackgroundImage の absolute inset-0
許可: ContentCard の absolute bottom-0
禁止: それ以外の要素への absolute の使用
```

ContentCard 内部のレイアウトは通常の `flex flex-col justify-between` で構成すること。

---

## 禁止事項
- 外部画像URL の使用禁止（Base64のみ）
- 絵文字の使用禁止（SVGアイコンで代替）
- ContentCard 内での `absolute` の使用禁止
- CTAボタンに `self-end` を使用禁止（`w-full` が仕様）
