# Layout Rules — 320×50 (Mobile Banner)

## レイアウトの型
**横一列（画像禁止）**

高さ50pxという極限の制約。画像は使用できない。
左側に SVGアイコン + キャッチコピー、右側に極小CTAを配置する横一列構造のみ許可。

---

## ルートコンテナ

```jsx
<div className="w-[320px] h-[50px] flex flex-row items-center justify-between px-3 py-2 bg-white">
```

---

## 左側（アイコン + コピー）

```jsx
{/* LeftBlock */}
<div className="flex flex-row items-center gap-2">

  {/* SVGアイコン（ブランドカラー） */}
  <svg width="18" height="18" viewBox="0 0 24 24" fill="none"
       stroke="var(--color-main)" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round">
    {/* コンテキストに合ったLucideアイコンのパスをここに記述 */}
    {/* 例: Star, Sparkles, Heart, Zap など */}
  </svg>

  {/* キャッチコピー — 15文字以内、truncate必須 */}
  <span
    className="text-[13px] font-semibold truncate max-w-[180px]"
    style={{color: 'var(--color-text)', fontFamily: 'var(--font-body)'}}
  >
    {catchCopy}  {/* 例: "春の無料体験、受付中" */}
  </span>

</div>
```

**重要**: `catchCopy` は必ず15文字以内に収めること。超過する場合は `truncate` でカット。

---

## 右側（極小CTA）

```jsx
{/* RightBlock */}
<button
  className="px-3 py-1 text-xs font-bold text-white whitespace-nowrap"
  style={{
    background: 'var(--color-accent)',
    borderRadius: '9999px',  /* rounded-full 相当 */
  }}
>
  {ctaText}  {/* 例: "詳しく" "予約" "無料" — 4文字以内推奨 */}
</button>
```

---

## 配置サマリー

| エリア | 幅 | 役割 | AIBAC |
|---|---|---|---|
| LeftBlock | flex-1（約220px） | SVGアイコン + コピー | Attention / Benefit |
| RightBlock | shrink-0（約80px） | 極小CTA | Action |

---

## コンテンツ文字数制限（厳守）

| 要素 | 上限 | 理由 |
|---|---|---|
| catchCopy | 15文字 | `truncate` で強制カット |
| ctaText | 4文字 | ボタン幅に収まる限界 |

---

## 禁止事項
- 画像（`<img>`, `background-image`）の使用禁止（高さ50pxに収まらないため）
- 絵文字の使用禁止（SVGアイコンで代替）
- 2行以上のテキスト禁止（`whitespace-nowrap` または `truncate` で1行に収める）
- フォントサイズ 10px 未満の使用禁止（視認不可）
